# petitmokus — Brand Asset Package

Production-ready logo files for web, email, and print.

## Folder structure

```
brand/
├── svg/                       Vector — preferred for web & print
│   ├── petitmokus-logo-color.svg              Primary mark, transparent bg
│   ├── petitmokus-logo-color-on-beige.svg     Primary on sandy backdrop
│   ├── petitmokus-logo-mono.svg               Single-tone (ink) variant
│   ├── petitmokus-logo-white.svg              White variant for dark bg
│   └── petitmokus-lockup-horizontal.svg       Mark + wordmark
│
├── png/                       Raster — when SVG isn't supported
│   ├── petitmokus-logo-{16,32,48,64,128,180,192,256,512,1024}.png   transparent
│   ├── email/                                 solid beige bg, email-safe
│   │   └── petitmokus-logo-email-{200,400,600}.png
│   ├── mono/                                  ink-on-transparent
│   │   └── petitmokus-logo-mono-{256,512,1024}.png
│   └── white/                                 white-on-dark with bg
│       └── petitmokus-logo-white-{256,512}.png
│
└── favicon/                   Browser tab + home-screen icons
    ├── favicon.ico            multi-resolution (16/32/48)
    ├── favicon.svg            modern browsers
    ├── favicon-16x16.png
    ├── favicon-32x32.png
    ├── favicon-48x48.png
    └── apple-touch-icon.png   180×180, solid beige bg
```

## Web — drop into the site `<head>`

```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
<meta name="theme-color" content="#F9F6F1">
```

For inline use, prefer the SVG:

```html
<img src="/brand/svg/petitmokus-logo-color.svg" alt="petitmokus" width="40" height="40">
```

## Email

Email clients (especially Outlook + Gmail dark mode) often invert or strip
transparency. Use the **PNGs in `png/email/`** which have a solid beige
background baked in:

```html
<img src="https://petitmokus.com/brand/petitmokus-logo-email-400.png"
     alt="petitmokus"
     width="120" height="120"
     style="display:block;border:0;outline:none;">
```

Recommended sizes:
- Email header: `200` or `400` rendered at `100–120px`
- Newsletter hero: `600` rendered at `200–240px`

## Usage guidelines

**Clear space.** Keep padding equal to the height of one ear (≈ 20% of the
mark's height) on all sides — no copy or imagery inside that zone.

**Minimum size.**
- Digital: `20px` square (favicons excepted — use the dedicated favicon files)
- Print: `12mm` square

**Don'ts.**
- Don't rotate, skew, or stretch the mark
- Don't recolor outside the palette above
- Don't drop shadows or apply effects
- Don't place the color mark on busy photography — use the white variant on a
  solid panel instead

**Accessibility.** Use `#5C4A3D` (Text dark) for any type paired with the mark
on light backgrounds — it passes WCAG AAA on every neutral in the palette. The
dusty rose `#D897A8` is decorative only; never use it for type.
